# Cache Replacement Policy Visualizer

An interactive visualization tool for studying cache replacement algorithms.

Implements FIFO, LRU, LFU, and OPT policies in C with a web-based visualization interface.

---

Features:
- Step-by-step cache simulation
- Neon-style visualization of cache blocks
- Access timeline with hit/miss indicators
- Random workload generator
- Policy performance comparison

---

# Installation and Excecution
- Clone the repository
- Install server depedencies
- Start the backend simulator
- Open the simulator, add in the values that need to be visualized and study the cache replacement algorithms!

---

# Learning Outcomes

- Implementing system-level algorithms in C
- Building an interactive visualization interface for algorithm behavior
- Understanding cache hits, misses, and memory locality concepts
- Integrating low-level C programs with a web-based interface using Node.js
